Flight Tracker Web Application

# Project Description
This application allows users to search for flights by airport, view flight details, visualize flight routes on a map, and save flights as 'favorites'.
Flight data is retrieved from the AviationStack API, and favorites are stored using SupaBase.

This application was designed for use on web browsers such as Safari (IOS), Microsoft Edge, and Google Chrome.
